# AmbitionBox Company Dataset Scraper

This project scrapes company data from AmbitionBox using Python and BeautifulSoup.

## Features
- Scrapes 10,000+ company profiles
- Extracts:
  - Company name
  - Rating
  - Reviews count
  - Industry
  - Location
  - Highly rated aspects
  - Critically rated aspects
- Handles pagination and missing values
- Exports dataset to CSV

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Pandas
- Numpy
- time
